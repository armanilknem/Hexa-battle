In menu state:
- Enter to go to Lobby state

In Lobby state:
- Enter to go to Play state
- Escape to go to Menu state

In Play State state:
- Enters Player Turn state (default)
- Escape to go to Pause state
- Enter to go to Enemy Turn State

In Enemy Turn state:
- Escape to go to Pause state
- Enter to go to Player Turn state

In Pause state:
- Escape to go to Player Turn state
- Backspace to go to Enemy Turn State
- Enter to go to Menu state
